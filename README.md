# Fraud-Transaction-Detection
Developed a machine learning model to detect fraudulent transactions.
In today's world with rapid growth of digital transactions , scam has became the serious concern. 
This project focuses on building a machine learning model that identifies where the transaction is fraud or not based on the financial data patterns.

**Features:**
- Detects fradulent transactions using machine learning techniques.
- Uses key features like transaction amount,and account balances.
- Identifies pattern associated with high risk behavior.
- Improves model performace through feature engineering.
- Evaluates results using precision ,recall,and F1-score.

**How it Works**
1. Collect and load the data.
2. Perfrom data cleaning and preprocessing.
3. EDA(Exploratory Data Analysis)
4. Feature enginnering.
5. Train machine leanring model on processed data.
6. Predict wheather a transaction if fraudulent or not.
7. Evaluate model performance.
   
 **Learnings** 
 - Gained practical experience of handling imbalance,noisy datasets.
 - Improved skills in data cleaning & Preprocessing.
 - Learned to use various evalation metrices.
 - Developed real-world insights into fraud pattern detection.
